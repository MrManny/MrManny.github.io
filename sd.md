# Stable Diffusion artist prompt modifiers

Almost all these images were generated with the following options:

| Option              |     Value |
|---------------------|----------:|
| Inference steps     |        50 |
| Dimensions          |   512x512 |
| Guidance scale      |       7.5 |
| Seed                |        42 |
| Upscaling GAN       |      none |
| Face correction GAN |      none |

Exceptions include:

- changed the seed for *Gustave Klimt*'s portrait, it was NSFW
- changed the seed for *Makoto Shinkai*'s portrait, as it was not a portrait
- changed the seed for *Thomas Kinkade*'s portrait, as it was not a portrait
- changed the seed for *Wes Anderson*'s portrait, as it was not a full portrait
